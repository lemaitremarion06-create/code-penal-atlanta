🚔 MISSION

L'Atlanta Police Department assure
la sécurité publique et le maintien
de l'ordre sur le territoire municipal.

━━━━━━━━━━━━━━━━━━━━━━

👮 MISSIONS PRINCIPALES

• Protection des citoyens ;

• Prévention de la criminalité ;

• Application des lois ;

• Interventions d'urgence ;

• Enquêtes judiciaires.

━━━━━━━━━━━━━━━━━━━━━━

🚓 UNITÉS

• Patrol ;

• Traffic Unit ;

• Criminal Investigation Division ;

• SWAT ;

• K9 Unit.

━━━━━━━━━━━━━━━━━━━━━━

⚖️ POUVOIRS

Les agents peuvent notamment :

• Contrôler une identité ;

• Procéder à une arrestation ;

• Effectuer une enquête ;

• Appliquer le Code pénal ;

• Exécuter un mandat.

🤝 COOPÉRATION

L'APD travaille avec :

• Department of Justice ;

• Bureau du Shérif ;

• Services d'urgence ;

• Autorités fédérales.

et les mesures d'urgence décidées
par les autorités compétentes.
