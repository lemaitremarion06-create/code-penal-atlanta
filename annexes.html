🚓 ANNEXE A — MDT OFFICIEL

Le MDT (Mobile Data Terminal) constitue le système officiel de consultation et de gestion des données policières et judiciaires.

Utilisation du MDT :

• Consultation des données ;
• Consultation des antécédents ;
• Gestion des dossiers ;
• Autorités habilitées ;
• Protection des données.

Sections MDT :

• Livre I & II ;
• Livre III & IV ;
• Livre V & VI ;
• Livre VII & VIII ;
• Livre IX & X ;
• Livre XI à XVII.

🔫 ANNEXE B — CLASSIFICATION DES ARMES

Objet :

La présente annexe définit les catégories d'armes autorisées, réglementées ou interdites sur le territoire de la Ville d'Atlanta.

Les dispositions du Livre X du Code pénal demeurent applicables.

━━━━━━━━━━━━━━━━━━━━━━

🔹 CLASSE A

Armes de poing à usage civil.

Exemples :

• Pistolets ;
• Revolvers ;
• Armes de défense autorisées.

Conditions :

• Permis valide ;
• Enregistrement obligatoire.

━━━━━━━━━━━━━━━━━━━━━━

🔹 CLASSE B

Armes longues civiles.

Exemples :

• Fusils de chasse ;
• Carabines ;
• Armes de tir sportif.

Conditions :

• Licence valide ;
• Transport sécurisé.

━━━━━━━━━━━━━━━━━━━━━━

🔹 CLASSE C

Armes réservées aux services gouvernementaux.

Exemples :

• Fusils d'assaut ;
• Mitraillettes ;
• Armes de service spécialisées.

Réservées aux :

• Atlanta Police Department ;
• Bureau du Shérif ;
• Services fédéraux ;
• Forces militaires.

━━━━━━━━━━━━━━━━━━━━━━

🔹 CLASSE D

Armes interdites.

Exemples :

• Lance-roquettes ;
• Lance-grenades ;
• Explosifs militaires ;
• Mines ;
• Armes modifiées ;
• Armes de destruction massive.

La possession de ces armes est interdite.

━━━━━━━━━━━━━━━━━━━━━━

📜 RÈGLES GÉNÉRALES

Classe A

• Autorisée avec permis.

Classe B

• Autorisée avec licence.

Classe C

• Réservée aux services autorisés.

Classe D

• Interdite ;
• Confiscation immédiate.

━━━━━━━━━━━━━━━━━━━━━━

⚖️ ANNEXE C — CIRCONSTANCES AGGRAVANTES ET ATTÉNUANTES

Objet :

La présente annexe précise les circonstances pouvant influencer les sanctions prévues par le Code pénal.

━━━━━━━━━━━━━━━━━━━━━━

🔺 CIRCONSTANCES AGGRAVANTES

Peuvent notamment être retenues :

• Utilisation d'une arme ;

• Participation en groupe ;

• Organisation criminelle ;

• Victime dépositaire de l'autorité publique ;

• Prise d'otage ;

• Récidive ;

• Mise en danger de plusieurs personnes.

━━━━━━━━━━━━━━━━━━━━━━

🔻 CIRCONSTANCES ATTÉNUANTES

Peuvent notamment être prises en considération :

• Coopération avec les autorités ;

• Aveux spontanés ;

• Première infraction ;

• Plaidoyer de culpabilité ;

• Réparation volontaire du préjudice.

━━━━━━━━━━━━━━━━━━━━━━

• Livre XVI — Récidive et application des peines.

🚨 ANNEXE D — PROTOCOLES DEFCON

Objet :

La présente annexe définit les niveaux d'alerte applicables sur le territoire de la Ville d'Atlanta.

━━━━━━━━━━━━━━━━━━━━━━

🟢 DEFCON 5

Situation normale.

Les services publics et les forces de l'ordre fonctionnent selon les procédures habituelles.

━━━━━━━━━━━━━━━━━━━━━━

🟡 DEFCON 4

Vigilance renforcée.

Une menace potentielle a été identifiée et les contrôles sont renforcés.

━━━━━━━━━━━━━━━━━━━━━━

🟠 DEFCON 3

Alerte élevée.

Les forces de sécurité sont mobilisées et des mesures exceptionnelles peuvent être mises en œuvre.

━━━━━━━━━━━━━━━━━━━━━━

🔴 DEFCON 2

Alerte critique.

Une menace imminente nécessite une mobilisation importante des autorités compétentes.

━━━━━━━━━━━━━━━━━━━━━━

⚫ DEFCON 1

État d'urgence.

Mobilisation générale et application des mesures exceptionnelles prévues.

━━━━━━━━━━━━━━━━━━━━━━

🏛 AUTORITÉS COMPÉTENTES

• Department of Justice ;
• Atlanta Police Department ;
• Bureau du Shérif ;
• Autorités municipales.

━━━━━━━━━━━━━━━━━━━━━━

🔄 RETOUR À LA NORMALE

Le niveau DEFCON est abaissé lorsque la menace a disparu et que la sécurité publique est rétablie.
