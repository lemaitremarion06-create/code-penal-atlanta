🚓 ANNEXE A — MDT OFFICIEL

Le MDT (Mobile Data Terminal) constitue le système officiel de consultation et de gestion des données policières et judiciaires.

Utilisation du MDT :

• Consultation des données ;
• Consultation des antécédents ;
• Gestion des dossiers ;
• Autorités habilitées ;
• Protection des données.

Sections MDT :

• Livre I & II ;
• Livre III & IV ;
• Livre V & VI ;
• Livre VII & VIII ;
• Livre IX & X ;
• Livre XI à XVII.

🔫 ANNEXE B — CLASSIFICATION DES ARMES

Les armes sont réparties selon les catégories suivantes :

• Classe A ;
• Classe B ;
• Classe C ;
• Classe D.

La réglementation applicable est définie par le Code pénal et les autorités compétentes.

⚖️ ANNEXE C — CIRCONSTANCES AGGRAVANTES ET ATTÉNUANTES

Circonstances aggravantes :

• Utilisation d'une arme ;
• Crime en bande organisée ;
• Victime dépositaire de l'autorité publique ;
• Récidive ;
• Prise d'otage.

Circonstances atténuantes :

• Coopération avec les autorités ;
• Aveux spontanés ;
• Première infraction ;
• Plaidoyer de culpabilité.

🚨 ANNEXE D — PROTOCOLES DEFCON

DEFCON 5 — Situation normale.

DEFCON 4 — Vigilance renforcée.

DEFCON 3 — Alerte élevée.

DEFCON 2 — Alerte critique.

DEFCON 1 — État d'urgence.

Dispositions :

• Autorités compétentes ;
• Déclenchement des protocoles ;
• Mesures exceptionnelles ;
• Retour à la normale.
